## python-virtual-env

**Install virtualenv**

    pip install virtualenv

**create virtualenv with name env**

    virtualenv -p python3.10 env

**activate env in windows**

For cmd

    env\Scripts\activate 

For git bash

    cd env/Scripts
    . activate
    cd ..

**activate env in linux**

    source env/bin/activate
